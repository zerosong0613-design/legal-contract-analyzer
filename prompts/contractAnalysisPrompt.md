## 등급 기준

### 리드타임 등급 (L)
| 등급 | 기준 | 검토 기간 |
|------|------|-----------|
| L1 | 표준·반복 계약, 소규모, 정형화된 조항 | 1일 내 |
| L2 | 비표준 조항 포함, 중간 규모, 협상 이력 있음 | 3일 내 |
| L3 | 복잡한 구조, 고액, 신규 거래 유형, 중대 IP/책임 이슈 | 협의 필요 |

### 리스크 등급 (R)
| 등급 | 기준 |
|------|------|
| R1 | 균형 잡힌 표준 조항, 양측 보호 장치 있음 (저위험) |
| R2 | 일부 불균형 조항, 수정 필요 항목 있음 (중위험) |
| R3 | 심각한 불균형, 중대한 리스크 조항, 즉각 협상·수정 필요 (고위험) |

### 리스크 severity
- `high`: 즉각 대응 필요
- `medium`: 검토 후 협상
- `low`: 참고 수준

### 주요 리스크 유형 예시
배상책임 / 지식재산권 / 해지조항 / 손해배상 / 비밀유지 / 독점조항 / 준거법 / 불가항력 / 계약금 / 품질보증
